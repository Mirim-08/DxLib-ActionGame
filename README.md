# DxLib-ActionGame
# 2D横スクロールアクションゲーム

## 概要

C++とDxLibを使用して制作した2D横スクロールアクションゲームです。

プレイヤーを操作してステージを進み、敵を踏みつけながらゴールを目指します。

## 開発環境

- C++
- DxLib
- Visual Studio 2022
- Windows 11

## 操作方法

| キー | 操作 |
|--------|--------|
| A | 左移動 |
| D | 右移動 |
| SPACE | ジャンプ |
| ENTER | ゲーム開始 |
| R | リスタート |
| ESC | 終了 |

## 実装機能

- タイトル画面
- 横スクロールステージ
- カメラ追従
- ジャンプ
- コヨーテタイム
- ブロックとの当たり判定
- 敵の巡回移動
- 敵の踏みつけ判定
- スコアシステム
- タイム計測
- ゲームオーバー
- ゲームクリア
- BGM
- 効果音

## 工夫した点

### コヨーテタイム

プレイヤーが足場から離れた直後でも一定時間ジャンプできるようにし、操作性を向上させました。

### カメラ追従

プレイヤー位置に応じてカメラを移動させ、横スクロールアクションゲームを実現しました。

### 敵との当たり判定

AABB判定を使用し、敵を上から踏んだ場合と横から接触した場合で処理を分けています。

### コード設計

Player、Enemy、Block、Background、Goal、Game、Titleにクラスを分割し、役割ごとに管理しています。

## 素材について
